# 🛡️ PANGLIMA Downloader

## ✨ Fitur

- ✅ Download Video TikTok No Watermark
- ✅ Download MP3 (Audio)
- ✅ Preview Thumbnail & Detail Video
- ✅ Copy Link dengan 1 Klik
- ✅ Paste dari Clipboard
- ✅ Progress Bar & Loading Animation
- ✅ Riwayat Download (Local Storage)
- ✅ Dark/Light Mode
- ✅ Responsive Mobile
- ✅ PWA Ready
- ✅ SEO Friendly
- ✅ Glassmorphism Design
- ✅ Particle Background

## 🚀 Quick Deploy ke Vercel

1. **Fork repo ini** ke GitHub Anda
2. **Klik tombol deploy** di atas atau:
3. Buka [Vercel](https://vercel.com)
4. Import repository dari GitHub
5. Tambahkan Environment Variable:
   - `API_KEY` = API Key TikTok Downloader Anda
